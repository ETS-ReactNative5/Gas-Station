VLC
=========

A generic role made for installing VLC on most platforms.

## Supported Platforms

* Linux
  * ArchLinux
  * CentOS 8
  * Fedora
  * Ubuntu 20.04
* Mac OS X
* Windows

## Example Playbook

```
- hosts: desktops
  roles:
    - vlc
```
