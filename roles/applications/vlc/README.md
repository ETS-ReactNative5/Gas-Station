Role Name
=========

A generic role made for installing VLC on most of the platforms.

Current platforms:
- Linux
  - CentOS 8
  - Fedora 32 
  - Ubuntu 20.04
  - ArchLinux
- Windows 10
- MacOS

Example Playbook
----------------
```yml
- hosts: servers
  roles:
    - vlc
```
